# JETHelper dictionary assets

JETHelper intentionally bundles only these reviewed Yomitan-compatible
archives:

- `jmdict_english.zip` — English vocabulary definitions and readings
- `kanjidic_english.zip` — kanji meanings, readings, and metadata
- `jiten_freq_global.zip` — global Japanese word-frequency ranks

Together they provide JETHelper's default vocabulary, kanji, and frequency
foundation.

Do not add personal or third-party dictionary archives to the release
allowlist without first deciding that they are appropriate to redistribute.
Additional dictionaries should normally be stored in a separate folder and
selected through `/jetconfig`.

See the repository-level
[`ACKNOWLEDGEMENTS.md`](../../../ACKNOWLEDGEMENTS.md) for data-source links,
attribution, related projects, and update information.
